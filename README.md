# app

## Develop environment require

- `POSIX` (bash, grep, cut, sort, etc)
- `nvm` https://github.com/nvm-sh/nvm
- `yarn` https://yarnpkg.com/getting-started/install

```shell
git clone project
cd project
nvm use
yarn install
```

## Develop

```shell
yarn dev
```

## Generate data and upload into ipfs

```shell
node util/dataUpload.js
```
